(x,y) のくみを全列挙して、その和にカウントを積むことで1回のループで全てnのカウントを求められる。
```md
count[x^2 + y^2] += 1 を全てのx < y について実行

→ 最後に count[i] == i のものが良い整数
```

## ループの打ち切り条件
```rust 
// x >= y になる前に終了
if x * x * 2 > n { break; }
```

`x < y` という制約から `x² < y²` なので `2x² < x² + y²` です。
つまり `2x² > n` になった時点で条件を満たす y は存在しないので打ち切れます。
```rust
if sum > n { break; }  // これ以上 y を増やしても n を超えるだけ
```

y を増やすほど sum は単調増加するので、超えた時点で内側のループを打ち切れます。

### 出力部分
```rust
let g: Vec<usize> = (1..=n).filter(|&i| count[i] == 1).collect();
```
filterで count == 1 のものだけ残して Vec に集めます。
```rust
let res = good.iter().map(|x| x.to_string()).collect::<Vec<_>>().join(" ");
```
数値を文字列に変換して join(" ") で空白区切りの1行にまとめます。

### 計算量
外側のループは x が `√(N/2)` まで回るので O(√N) 回、内側は y が最大 √N まで回るので合計 `O(N)` 回です。
